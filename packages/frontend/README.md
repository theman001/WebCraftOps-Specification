# frontend 패키지

WebCraftOps 웹 프런트엔드의 1단계 뼈대를 포함합니다. 서버 선택 화면에서 Bridge 연결 테스트와 최근 서버 기록을 제공합니다.

## 포함 기능 (초기)

- Bridge URL 입력 및 연결 테스트
- 최근 서버 기록 (LocalStorage)
- 청크 바이너리 디코드 샘플 (Web Worker)
- Edit Job 생성/조회/상태 변경 UI
- Edit Job Undo/Redo 기록 추적 UI
- Blueprint/Schematic 업로드 및 붙여넣기 작업 생성 UI
- 감사 로그 대시보드 UI (필터/조회 개수 포함)
