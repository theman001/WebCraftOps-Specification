Bridge 서버 주소 입력

Bridge URL을 입력하고 연결 테스트를 진행하세요.

백엔드가 브릿지 프록시를 제공하면 CORS 없이 레지스트리를 가져옵니다.
대기 중입니다.
최근 연결

    레지스트리 블록 팔레트

    Bridge 연결 후 팔레트를 불러옵니다.

    청크 이진 디코드 (샘플)

    Bridge에서 청크 바이너리를 가져와 Web Worker로 디코드합니다.

    대기 중입니다.

      Edit Job 시스템 (Command 패턴)

      Job 생성/조회 및 일시정지/재개/취소를 UI에서 확인합니다.

      Bridge 연결 후 작업을 생성할 수 있습니다.

      Undo/Redo 작업 기록

      상태 변경 내역을 기록하고 마지막 작업을 되돌리거나 재실행합니다.

      기록이 없습니다.

        Blueprint/Schematic 관리

        schem 파일을 등록하고 작업 큐에 붙여넣기 커맨드를 생성합니다.

        파일을 선택하면 메타데이터를 등록합니다.

        감사 로그 대시보드 (준비)

        작업 큐 실행 로그를 조회해 감사 대시보드 기반을 점검합니다.

        조회 대기 중입니다.
