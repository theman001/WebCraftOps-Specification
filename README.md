# WebCraftOps 사양 저장소

WebCraftOps는 웹 기반 Minecraft Ops 콘솔, 3D 월드 에디터, 블루프린트/스키매틱 워크플로우를 통합한 플랫폼을 목표로 합니다. 이 저장소는 AI 에이전트와 인간 개발자를 위한 설계 사양을 문서화합니다.

## 문서 안내

- [WebCraftOps 사양 v0.3](docs/spec-v0.3.md)

## 구성 요약

- 설계 원칙: 서버로부터의 진실, 버전 불가지, 모드 로더 분리
- 핵심 기능: 서버 운영, 3D 월드 편집, 블루프린트 워크플로우
- 기술 포인트: 레지스트리 덤프 2.0, 이진 청크 전송, 작업 큐/커맨드 패턴, 적응형 스로틀링
- 보안 모델: 온라인 모드 기반 OAuth 및 권한 어댑터

## 사용 방법

사양을 확인하려면 상단의 v0.3 문서를 열어 주세요. 구현 및 검토는 해당 문서를 기준으로 진행합니다.
